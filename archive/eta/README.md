# Eta calibration — moved to tariff-etr-adj

sibling repo **tariff-etr-adj**, which delivers the production
`eta_schedule_hs2_by_country.csv` to the Tariff Model. The copy here
(`08_eta_calibration.R`, last run against this repo's data 2026-06) is
archived to avoid two drifting implementations — do not extend it.
